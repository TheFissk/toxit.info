    Controls / Operation

    Welcome!

    Welcome to toxit

    Toxit is an explorative data science esque web application that centers around the exploration of an undirected weighted graph of various communities gathered from reddit.com By configuring our gather bot service to go and collect snapshots for us we are able to create different snapshots of data that consist of a set of subreddit commnities, and the common moderators and commentors between those communities. Each of these communities shows a single value under the name indicating the number of comments flagged by our AI when generating its average toxiticy score. This allows users to navigate through different snapshots of data and explore the connections between communities to reveal intersting trends or to just have fun playing with the physics of the graph.

    Graph Exploration Controls

    Click a node to select it. By default, for easy navigation, this opens the navigation tab with information about its relations to other nodes. You can also drag and drop nodes to move them around. Hover over a node to view its detailed toxicity data.

    Middle Mouse button &

    When middle mouse button is clicked or the button at the top right corner of the screen is clicked (left mouse click) the currently loaded graph will zoom the node to make it fit nicely in the screen, making a nice reset feature if you ever need to return the graph to a more neutral state.

    Side Menu

    By default the side menu is in a collapsed state. When the user clicks the it will open the side menu.

    Drag & Drop Sidebar Tabs

    Sidebar tabs with the symbol can be reordered.

    Snapshot Selection

    Since subreddits change over time, we compare them against each other at a particular moment in time. You can explore different snapshots using the dropdown. You can also toggle between common moderators and commenters when viewing connections.

    Theme & style

    Change the website styling to use a dark color theme or bright color theme with orange accents. The default styling is for darkmode.

    Node Information

    Provides toxicity data of a node selected by clicking on it as well as a link to the subreddit. Contains stastically information like average value, min and max values, as well as standard deviation.

    Physics

    Provides the abilitiy to customize node graph appearance.

    Export

    Download the the selected snapshot data in the specified format; The options are Json, CSV, and Pickle or PIC.
