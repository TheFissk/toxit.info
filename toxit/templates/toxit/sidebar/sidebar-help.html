    Controls / Operation

    Welcome!

    Welcome to toxit.info

    Toxit is a data exploration platform that centers around a weighted undirected graph representation of Reddit.com community connections. Graph edges are weighted by set intersection cardinality of either moderators or commentators for the snapshot inder investigation. Comments are passed to a natural language processing model which returns the hate speech scoring of the comment. Our NLP model was trained on The Measuring Hate Speech Corpus to within 0.1132 MAE over [-1,1].

    Graph Exploration Controls

    Click a node to select it. By default, for easy navigation, this opens the navigation tab with information about its relations to other nodes. You can also drag and drop nodes to move them around. Hover over a node to view its statistics.

    Middle Mouse button &

    When middle mouse or is clicked the graph will reset.

    Side Menu

    By default the side menu is in a collapsed state. When the user clicks the it will open the side menu.

    Drag & Drop Sidebar Tabs

    Sidebar tabs with the symbol can be reordered.

    Snapshot Selection

    Data samples change over time, You can explore different snapshots using the dropdown. You can also toggle between common moderators and commenters when viewing connections.

    Theme & style

    Change the website styling to use a dark color theme or bright color theme with orange accents. The default styling is for darkmode.

    Node Information

    Provides descriptive statistics of a node selected by clicking on it as well as a link to the subreddit.

    Physics

    Customize graph physics.

    Export

    Download as JSON, CSV, or Pickle.
