    Middle Mouse button &

    When middle mouse button is clicked or the button at the top right corner of the screen is clicked (left mouse click) the currently loaded graph will zoom the node to make it fit nicely in the screen, making a nice reset feature if you ever need to return the graph to a more neutral state.

    Side Menu

    By default the side menu is in a collapsed state. When the user clicks the it will open the side menu.
